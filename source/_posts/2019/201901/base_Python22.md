#### 练习
    利用切片实现去除首尾空格
- 代码
    ```python
        def pythonForTrim(str):
            while str[:1] == ' ':
                str = str[1:]
            while str[-1:] == ' ':
                str = str[:-1]
            
            return str

        str = '  '
        print(len(str))
        str = pythonForTrim(str)
        print(len(str))

        str1 = ' 1234  '
        print(len(str1))
        str1 = pythonForTrim(str1)
        print(len(str1))

        # 输出结果
        2
        0
        7
        4
    ```
